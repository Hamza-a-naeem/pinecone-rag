# Troubleshooting Guide

## Installation Issues

### Problem: Installation fails with error code 0x80070643
**Solution:**
1. Uninstall any previous versions
2. Clear temporary files from C:\Temp
3. Run installer as administrator
4. Ensure .NET Framework 4.8 is installed

### Problem: "Unable to connect to server" error
**Solution:**
1. Check internet connection
2. Verify firewall settings allow port 443
3. Try connecting to https://status.techcorp.com to check service status
4. Clear DNS cache: `ipconfig /flushdns`

## Sync Issues

### Problem: Files not syncing
**Solution:**
1. Check available storage space
2. Verify file names don't contain special characters: `< > : " / \ | ? *`
3. Ensure file is not locked by another application
4. Check sync status in settings panel

### Problem: Sync speed is slow
**Solution:**
1. Check network bandwidth
2. Pause other active downloads
3. Try syncing during off-peak hours
4. Contact support if issue persists

## Login Issues

### Problem: Forgot password
**Solution:**
1. Go to https://app.techcorp.com/reset
2. Enter your email address
3. Check email for reset link (check spam folder)
4. Link expires in 1 hour

### Problem: Two-factor authentication not working
**Solution:**
1. Ensure device time is synchronized
2. Try backup codes from initial setup
3. Contact support to disable 2FA temporarily
4. Re-enable 2FA after regaining access

## Performance Issues

### Problem: Application running slowly
**Solution:**
1. Close unnecessary applications
2. Check system resources (CPU, RAM, Disk)
3. Update to latest version
4. Clear application cache in Settings > Advanced
5. Restart the application
